wing 7/21/2021 在线文档

集合对象（数组、对象相互嵌套），操作时总是递归，此插件就此运用而生

# vue安装

通过 npm引入：

$ npm i fair-wing
1

vue注册：

import Fair from 'fair-wing'
Vue.use(Fair)
1
2

使用说明：

为了避免和vue内置方法或其他插件及公共方法名称冲突,在注册插件向vue上绑定方法时，将该插件的名称进行了特殊的区分 ，例如find()方法需要在其前面添加this.$F_

// find()应写为
this.$F_find()

2

若不想在vue上注册方法，也可以在组件内单独引用直接使用

import { find } from 'fair-wing'
find()

# find系列

# find

查找源对象

参数

  • find(obj, second, third, fourth)
    obj:查找的源对象
    second: 对象{},值1
    third: 键名, 层级
    fourth:层级(有层级时，遍历到此层级就不会再向下遍历，节省性能)

返回值

{
    by: {key: val}, // 通过查找的条件
    own: item, // 包含键名的当前对象
    parent: parent, // 父对象
    parents: [null, ... , {id: 123}], // 祖先们
	  level: level // 层级，从0开始
}

例子

let obj = {
        id: 1,
        a: 1,
        b: {
          id: 2,
          d: 4
        },
        c: [1,2,3, {id: 2, c: 3}]
      }
find(obj, 123, 'id', 2) // 源对象, 键值, 键名, 层级(可不传)
find(obj, {id: 123}, 2) // 源对象, {键名: 键值}, 层级(可不传)

/*

# addOnlykey

向集合对象中的每个对象或数组添加唯一值（Symbol(117fa))

参数

  • addOnlykey(obj, key = 'id')
    obj: 要添加唯一值的源对象
    key: 要添加的key(默认id)

返回值

  • 添加唯一值的集合对象

例子

let arr = [1, {a: 1}, []]
let arrs = addOnlykey(arr, 'ids')
console.log(arrs)
/*
[
	1, 
	{a: 1, ids: Symbol(1743e)},
	[ids: Symbol(13359)], 
	ids: Symbol(117fa)
 ]
*/

# delKey(obj, key)

将集合对象中的特定的key强行全部删除，通Object.definePrototype设置的也一样会被删除

参数

  • delKey(obj, key)
    obj: 要删除的源对象
    key: 要删除的key

返回值

  • 删除特定key后的集合对象

例子

let arr = [1, {a: 1, ids: Symbol()}]
let arrs = delKey(arr, 'ids')
console.log(arrs) // [1,{a: 1}]

# forEach

像遍历数组一样遍历集合对象，遍历集合对象(像es6的forEach一样不能停止)。

此方法有很大的拓展性, find系列方法均由此方法完成,但是find方法中做了性能优化,所以能满足需求时推荐使用find系列。

参数

  • forEach(obj, fn(item, key, parent, level)) obj:查找的源对象

    fn(value, key, parent, level): 回调函数
    value: 遍历到的当前key值
    key: 遍历到的当前key
    parent: 遍历到的当前对象
    level: 遍历源对象的当前层级

无返回值
